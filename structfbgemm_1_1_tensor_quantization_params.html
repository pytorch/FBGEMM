Detailed Description

Struct from gemmlowp

A structure to hold quantization parameters scale and zero_point. The meaning of these values is as the constants in the quantization equation

real_value = scale * (quantized_value - zero_point)

In other words, 'zero_point' is the quantized value that corresponds to the real value 0, and 'scale' is the difference of real values corresponding to consecutive quantized values.
