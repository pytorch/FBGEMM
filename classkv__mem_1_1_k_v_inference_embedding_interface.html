Detailed Description

template<typename weight_type>
class kv_mem::KVInferenceEmbeddingInterface< weight_type >

This interface defines the core API that all KV embedding implementations must provide, enabling different backend implementations (DRAM, SSD, etc.) to be used interchangeably.
