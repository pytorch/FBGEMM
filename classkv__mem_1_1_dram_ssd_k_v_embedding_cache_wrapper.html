Detailed Description

This wrapper is the sole initializer of the DRAM+SSD backend: it builds the DRAM (L2) cache, constructs the SSD (L3) RocksDB tier internally, and assembles the composite DramSsdKVEmbeddingCache from both. The wrapper layer keeps a single DRAM-backed composite (impl_) and a single RocksDB storage (rocksdb_impl_); the weight-type (Half/float) selection is confined to the templated init_composite() helper. Because RocksDB is always created in the constructor, rocksdb_impl_ is guaranteed non-null for all checkpoint/snapshot code paths.
